# Restaurant Order Management System

## Purpose 
This is a Python program that simulates a restaurant menu and order system.  
You can see the menu, add items to a cart, calculate the bill, track inventory, and check daily sales.

## Features
- Show menu with categories (Starters, Mains, Desserts)  
- Add and remove items from your order (cart)  
- Calculate subtotal, GST, and total bill  
- Track inventory and show alerts when stock is low  
- Analyze sales data to find the best-selling day and most ordered items  

## How it works
1. The menu, inventory, and sales data are stored using dictionaries and lists.  
2. You can add items to a cart, remove items, or update quantities.  
3. When an order is placed, inventory is updated.  
4. The program can show daily revenue and analyze sales trends.
